Upload CSV Files
  • Create a comma seperated file with your password data.
  • A program like Excel or OpenOffice can help edit this file.
  • The first row should be the header. It should consist of 3 or optionally 4 columns. Like:
    • site,username,password,notes
    • or
    • site,username,password
  • Each line of the comma seperated file should have the same number of columns as the header.
  • After import, the file is deleted from the server. All entries are stored fully encrypted.
